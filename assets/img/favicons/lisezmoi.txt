Pack Standard + Extra de favicons (pour toutes les versions actuelles des principaux navigateurs, iOS, Android et Internet Explorer 10+)

* Vous pouvez ajouter plusieurs favicons à votre page Web. Pour cela, copiez l'ensemble des fichiers téléchargés à la racine de votre site Web et insérez les balises HTML suivantes dans le code source de votre page web, après la balise <head> et avant la balise </head> :

<link rel="shortcut icon" href="favicon.ico" type="image/x-icon">
<link rel="icon" sizes="16x16" href="favicon-16.png" type="image/png">
<link rel="icon" sizes="32x32" href="favicon-32.png" type="image/png">
<link rel="icon" sizes="48x48" href="favicon-48.png" type="image/png">
<link rel="icon" sizes="96x96" href="favicon-96.png" type="image/png">
<link rel="icon" sizes="192x192" href="favicon-192.png" type="image/png">
<link rel="apple-touch-icon" sizes="57x57" href="apple-touch-icon-57x57.png">
<link rel="apple-touch-icon" sizes="60x60" href="apple-touch-icon-60x60.png">
<link rel="apple-touch-icon" sizes="72x72" href="apple-touch-icon-72x72.png">
<link rel="apple-touch-icon" sizes="80x80" href="apple-touch-icon-iphone.png">
<link rel="apple-touch-icon" sizes="114x114" href="apple-touch-icon-114x114.png">
<link rel="apple-touch-icon" sizes="120x120" href="apple-touch-icon-120x120.png">
<link rel="apple-touch-icon" sizes="144x144" href="apple-touch-icon-144x144.png">
<link rel="apple-touch-icon" sizes="152x152" href="apple-touch-icon-ipad.png">
<link rel="apple-touch-icon" sizes="167x167" href="apple-touch-icon-ipad-retina.png">
<link rel="apple-touch-icon" sizes="180x180" href="apple-touch-icon-iphone-retina.png">
<link rel="manifest" href="/manifest.json">
<meta name="msapplication-TileImage" content="ms-favicon-144.png">
<meta name="msapplication-TileColor" content="#FFFFFF">
<meta name="theme-color" content="#ffffff">

* Optionnel: si vous souhaitez afficher le favicon animé, copiez le fichier animated_favicon.gif à la racine de votre site Web et insérez la balise HTML suivante dans le code source de votre page web, après la balise <head> et avant la balise </head> :

<link rel="icon" href="animated_favicon.gif" type="image/gif">
	
* Les fichiers suivants sont inclus dans votre pack :

- favicon.ico - favicon multi-resolution (16x16 to 48x48).
- favicon-16.png - 16*16 fichier de favicon des navigateurs standards au format PNG.
- favicon-32.png - 32*32 fichier de favicon des navigateurs standards au format PNG.
- favicon-48.png - 48*48 fichier de favicon de Google et Android au format PNG.
- favicon-96.png - 96*96 fichier de favicon de Google au format PNG.
- favicon-192.png - 192*192 fichier de favicon de Google et Android au format PNG.
- favicon-256.png - 256*256 fichier de favicon de navigateurs au format PNG.
- favicon-512.png - 512*512 fichier de favicon au format PNG.
- animated_favicon.gif - version animée du favicon au format GIF.
- apple-touch-icon-57x57.png - 57*57 fichier de favicon pour iPhone et iPod au format PNG.
- apple-touch-icon-60x60.png - 60*60 fichier de favicon pour iPhone au format PNG.
- apple-touch-icon-72x72.png - 72*72 fichier de favicon pour iPad au format PNG.
- apple-touch-icon-iphone.png - 80*80 fichier de favicon pour iPhone au format PNG.
- apple-touch-icon-114x114.png - 114*114 fichier de favicon pour iPhone 4 au format PNG.
- apple-touch-icon-120x120.png - 120*120 fichier de favicon au format PNG.
- apple-touch-icon-144x144.png - 144*144 fichier de favicon au format PNG.
- apple-touch-icon-ipad.png - 152*152 fichier de favicon pour iPad au format PNG.
- apple-touch-icon-ipad-retina.png - 167*167 fichier de favicon pour iPad retina au format PNG.
- apple-touch-icon-iphone-retina.png - 180*180 fichier de favicon pour iPhone retina au format PNG.
- ms-favicon-144.png - 144*144 fichier de favicon au format PNG.
- ms-smalltile.png - 70*70 fichier image de favoris personnalisée pour IE11 au format PNG.
- ms-mediumtile.png - 150*150 fichier image de favoris personnalisée pour IE11 au format PNG.
- ms-largetile.png - 310*310 fichier image de favoris personnalisée pour IE11 au format PNG.
- browserconfig.xml - fichier XML de configuration du navigateur pour définir une image de favoris par défaut sous IE11.
- manifest.json - Le but du manifeste est d'installer des applications Web sur l'écran d'accueil d'un appareil, offrant aux utilisateurs un accès plus rapide et une expérience plus riche.
- lisezmoi.txt - Ce guide d'installation rapide.
